实用代码¶

以下代码有助于进行测试。

开启全制造¶

c_freecrafting()

跳过一段时间¶

LongUpdate(100) -- 跳过100秒
LongUpdate(500, 1) -- 跳过500秒, 且玩家三围不受到影响

模组专用作弊码¶

以下为本模组新增的专用作弊码，在使用这些代码前，需先执行： 

require "momo_debug" -- 开启作弊码

获得桃的食物¶

m_food()

执行该指令后会获得四种桃色失败料理、暗堕稳定剂、蛋白质、碎冰冰和运动运料。

获得桃的专属物品¶

m_item()
执行该指令后会获得吉祥物帽子、变身杖、攻击杖、“夜色”和暗堕稳定剂。

召唤梅塔子¶

m_pet()

执行该指令后会生成一个梅塔子，并绑定为你的宠物。

切换皮肤¶

m_skin(1) -- 切换到常服
m_skin(2) -- 切换到蜜柑的衣服
m_skin(3) -- 切换到战斗服
m_skin(4) -- 切换到暗堕形态

执行该指令可以切换皮肤外观，但不影响角色机制。

例如，在暗堕状态下，输入m_skin(1)可以将外观切换到常服，但魔力流失效果和暗影武器加成仍然存在。

修改魔力¶

momo_magic.current = 150

立刻将桃的魔力设置为150。

测试版更新日志¶

2022-7-21 21:00 （最新）¶

  • 秘制蛋白饮品现在被判定为肉类。
  • 为攻击杖添加了命中特效。

2022-7-20 20:00¶

  • 修正了牛背上的动画，包括魔杖蓄力和掰断棒冰。
  • 碎冰冰现在有20天的腐败时间；碎冰冰和半根碎冰冰在冰箱中不会继续融化。
  • 修复了暗堕状态下魔法棒仍然可以蓄力的bug。

2022-7-19 19:00¶

  • 变身杖现在只能由桃装备。
  • 在蓄力时桃会始终面向攻击目标。
  • 新增了一种鸡蛋豆芽可能种出的植物。
  • 调整了食物的制作配方。
  • 调整了蛋白饮品buff的持续时间和效果。

2022-7-19 8:00¶

  • 修复其他角色进入游戏的崩溃。

2022-7-18 22:00¶

  • 桃现在可以投掷哑铃。
  • 略微缓解了银制魔法棒攻击离开射程的目标时哑火的问题（未确认）。
