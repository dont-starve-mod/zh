爆炸是一门艺术¶

晓美焰可以制作各种爆炸类武器，这类武器具备范围效果，可攻击多名敌人。但也要小心误伤自己和他人！

太长不看版：

  • 雷管：制作爆炸类武器的必备材料。
  • 手雷：伤害不俗，还能摧毁建筑。
  • 莫洛托夫鸡尾酒：稳定的持续范围伤害。
  • 闪光弹：控场，消除仇恨，坑队友。
  • 刺突爆雷：近战武器，威力巨大，但也会波及到使用者。
  • 火箭发射器：远程打击，需要装弹。
  • 火箭助推榴弹：用于装填火箭发射器。

雷管¶

c_give("homura_deconator", 10)
  • 制造手雷的必需材料。
  • 配方：火药x1，金块x1 -> 5份。
  • 被点燃后会引爆，造成轻微的伤害和摧毁效果。

破片手雷¶

c_give("homura_bomb_bomb", 10)
  • 落地时立即爆炸，造成200范围伤害。
  • 配方: 雷管、燧石x4。

莫洛托夫鸡尾酒¶

c_give("homura_bomb_fire", 10)
  • 制造一片火海，呆在火中的生物会受到巨量烧伤。
  • 配方: 雷管、木炭x3。
  • 燃烧时间：25-30秒。

闪光弹¶

c_give("homura_bomb_flash", 10)
  • 配方: 雷管、硝石x1。
  • 引爆时发出强光，消除周围生物的攻击仇恨，并让它们陷入短暂的致盲状态。

  • 闪光弹对玩家也有效，但只要背对引爆位置就不会被致盲。

  • 闪光弹会对暗影生物造成50伤害。

刺突爆雷¶

c_give("homura_stickbang")
  • 近战武器，击中目标时引发爆炸，造成250范围伤害和摧毁效果，同时对使用者造成20伤害。
  • 配方：雷管x1，树枝x1，蜂刺x1。
  • 在静止的时间中，刺雷不会爆炸。
  • 手持刺雷跑动，玩家进入冲刺状态，移速提高。
  • 在冲刺状态下碰撞到任何可攻击目标，都会引爆刺雷。

AT4火箭筒¶

c_give("homura_rpg")

  • 范围攻击的远程武器，威力非常大，但会点燃攻击目标。
  • 配方：火箭助推榴弹x1，鳞片x1，火焰吹箭x1。
  • 配件槽：1。一般来说，对火箭筒最有用的配件是制冷装置（移除点燃效果）和四次元的弹夹（增加弹夹容量）。
  • 弹夹容量：1。
  • 基础伤害：300。

火箭弹¶

c_give("homura_rpg_ammo1", 10)
  • 用于装填火箭发射器。
  • 配方：雷管x5，木头x2，榴莲x1。 -> 5发
