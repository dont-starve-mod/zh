独行长路
联机, 辅助
单人联机延迟消除工具。完美解决延迟问题，无论存档有没有洞穴，操作都丝般顺滑。
千代田桃
联机, 人物
以碎冰冰之名起誓！
Sakana & Chinanago
联机, 装饰
可爱捏。
晓美焰
联机, 人物, 物品
掌握了时间魔法的少女。发动能力可以暂停周围所有物体的时间流逝。但是这个技能本身不具备攻击性，所以她用各种枪械辅助战斗。
错误追踪（不维护）
联机, 辅助, API
一个有助于定位mod错误位置的辅助工具。
同时为广大mod开发者提供了一个简洁的接口，方便玩家将报错日志发送到开发者手中。
不要点点点
联机, 界面美化

魔法少女小圆（不维护）
