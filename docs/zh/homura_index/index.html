简介

晓美焰是一个饥荒联机版的人物mod。包含了一位新角色和一系列新武器。

这是我个人最满意的一个作品。

订阅mod

直接下载mod文件

魔法弓¶

这是一把真正的弓箭。 查看详情

热武器¶

十几种各具特色的现代热武器，灵活使用这些武器来对抗敌人。 查看详情

2022-3月更新内容¶

本次更新主要是为了适配饥荒联机版的生活质量更新，同时还包括一些小优化。

  • 【重要改动】无烟火药被移除了
    • 所有需要无烟火药的武器配方现在均改为普通火药
    • 旧存档内的无烟火药会自动转变为普通火药
    • 新增一个配方将粉末蛋糕转换为火药

  • 【兼容】

    • 添加了“军火”图标以适配官方新版制造栏

  • 【修复】“假日”的水枪攻击动作现已正常

  • 【优化】移除了爆炸类武器的血量属性，雷管/手雷/燃烧瓶/闪光弹不再会被意外引爆
  • 【优化】对支援信标进行了一些改进
    • 支援信标共3个等级，但本身不再可以升级
    • 1级信标的空投掉落物中有几率出现2级信标蓝图，学习后可直接建造2级信标
    • 2级信标的空投掉落物中有几率出现3级信标蓝图，学习后可直接建造3级信标
    • 3级信标的空投掉落物中有几率出现武器设计图纸，包括刺突爆雷，寒冰射手，“假日”，饥饿之眼
    • 降低了连续获得同种武器设计图纸的概率，因此上述四种武器图纸更容易凑齐了
    • 空投不再会到有物理体积的障碍物附近
    • 支援信标最开始的5次激活必掉一张蓝图（俗称保底）
    • 修复支援信标充能通知异常的bug
  • 【优化】现在完全充能的魔法箭对围墙只造成轻微的伤害，这是为了防止超远射程的魔法箭掠过几百米开外的基地后造成意外的破坏

2022-2月更新内容¶

角色特性¶

  • 【新增专属武器-魔法弓】可以蓄力攻击的远程武器。这是一个隐藏物品，你必须先把模组设置中「重塑后的世界」一项开启，才能制作魔法弓。（详情）

热武器¶

  • 【改动】所有角色都能使用武器工作台了！
    • 晓美焰可以搭建工作台，并在工作台边编写“工作台操作手册”。
    • 其他角色在认真阅读手册后就能学会各种热武器的制作方法。
    • 晓美焰制作热武器的速度比常人更快。
  • 【改动】所有枪械武器和火箭炮现在都能在牛背上发射和装填。
  • 【新增机制】高级武器工作台可以接受特殊武器的图纸。（见支援信标）
  • 【新增武器】
    • 刺突爆雷：手持爆炸性武器，手持该武器跑动时进入冲刺状态，碰撞时会爆炸。
    • 寒冰射手：散发出阵阵寒气，被击中的敌人动作变得更加缓慢。
    • 饥饿之眼：类似泰拉瑞亚联动中的眼面具和盾牌，只要喂食物就能填充耐久。
    • “假日”：这是一把水枪，虽然是水枪，但打人可是很疼的。

建筑¶

  • 【改动】初级武器工作台的制作变得更容易，高级工作台变得更难；扳手的制作材料修改为月岩和噩梦燃料。
  • 【新增建筑-支援信标】激活后可以呼叫一次空投，获得免费的物资，只能造一座。
  • 【新增道具-支援信号枪】远程激活信标，在世界的任何一个角落呼叫空投。（制作中...）
  • 【新增机制】空投内有一定几率获得特殊武器图纸，把图纸交给工作台，就能解锁制作这些特殊武器:
    • 刺突爆雷
    • 寒冰射手
    • 饥饿之眼
    • “假日”

其他小改动¶

  • 【优化】枪械类武器的操作手感更加丝滑和稳定。
  • 【改动】下调了部分配件的科技需求。
  • 【修复】狙击枪无子弹时开镜，角色会有语言提示。

模组自定义配置¶

设定 说明
语言 设置模组语言，默认和游戏语言设定一致，可强制设定为中文或英文。
技能键 设置时停的快捷按键，默认 L。
填弹键 设置自动换弹的快捷按键，默认 R。
重塑后的世界 开启后，晓美焰无法发动时间停止，但可以使用魔法弓。
友伤 手雷和火箭弹的范围伤害不分敌我，默认开。关闭后就不能用手雷炸队友了。
PVP模式下此设定无效。
时停特效 发动时间停止时的特殊视觉效果，可以选择向外扩散的冲击波或仅仅让场景变灰。
